# Basis

Bekijk een voorbeeld van een basispagina in:

- [Storybook](https://nl-design-system.github.io/rvo/?path=/story/pagina-s-templates-basispagina--default)
- [Een volledig scherm binnen Storybook](https://nl-design-system.github.io/rvo/iframe.html?args=&id=pagina-s-templates-basispagina--default&viewMode=story)

De basispagina van een applicatie of website bevat minimaal de volgende onderdelen:

- Het Header component met daarin het logo.
- Een content vlak met een maximale breedte.
- In het content vlak staat minimaal één Header component van het type h1 met de paginatitel.
- Een Footer component met de verplichte onderdelen.
