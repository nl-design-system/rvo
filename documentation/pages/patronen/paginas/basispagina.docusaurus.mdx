[Bekijk een voorbeeld](https://nl-design-system.github.io/rvo/iframe.html?args=&id=pagina-s-templates-content--default&viewMode=story) van een basispagina in een nieuwe tab.

De basispagina van een applicatie of website bevat minimaal de volgende onderdelen:

- Het Header component met daarin het logo.
- Een content vlak met een maximale breedte.
- In het content vlak staat één Header component van het type h1 met de paginatitel.
- Een Footer component met de verplichte onderdelen.
