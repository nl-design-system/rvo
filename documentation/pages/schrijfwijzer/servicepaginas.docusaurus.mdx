# Servicepagina's

_Laatst gecontroleerd: 27 juni 2025_

## Inleiding

Iedere RVO-gerelateerde website of online middel moet dezelfde set aan servicepagina's hebben met dezelfde tekst. De servicepagina's zijn op de website via de footer te bereiken. Zo zorgen we voor duidelijk afzenderschap. Hoe de footer eruit moet zien, vind je op de pagina <a href="https://nl-design-system.github.io/rvo/docs/componenten/footer/" class="rvo-link">Footer</a>.

De volgende servicepagina's zijn in onderstaande volgorde beschikbaar:

- Toegankelijkheid
- Privacy
- Cookies
- Kwetsbaarheid melden
- Copyright
- Archief

### Instructies

Per servicepagina zijn er korte instructies om de tekst op de website te plaatsen. Volg daarnaast de volgende algemene instructies:

- Volg de toegankelijkheidsrichtlijnen.
- Begin elke pagina met een H1-kop (bijvoorbeeld Archief) en gebruik H2-subkoppen.
- De eerste alinea is de introtekst. Zorg ervoor dat deze de opmaak krijgt van een introductieparagraaf. Maak de tekst niet op via 'bold' of 'strong', maar gebruik de alineastijl voor introductie-paragrafen.

### Vragen en opmerkingen?

Neem contact op met Webportfoliomanagement, Regie en Advies via webportfoliomanagement@rvo.nl.

## Toegankelijkheid

### Instructies

- Kopieer en plak onderstaande tekst op de servicepagina. Voor Nederlandse websites is de URL domein.nl/toegankelijkheid. Voor Engelse websites geldt domein.nl/accessibility.
- Plaats in de footer een link naar deze pagina met als linktekst 'Toegankelijkheid' of 'Accessibility'.
- Het toegankelijkheidslabel is een afbeelding die via een embedded code op de pagina geplaatst wordt. De embedded code is aan te vragen via webportfoliomanagement@rvo.nl. Er bestaat geen Engelstalige variant van het toegankelijkheidslabel. Daarom plaatsen we die niet op Engelse websites.
- Op de pagina Accessibility is de tekst 'Read our accessibility declaration (in Dutch).' de link naar de toegankelijkheidsverklaring.
- Het is niet nodig om het pdf-bestand van het toegankelijkheidsonderzoek als download aan te bieden. Het rapport is in te zien via de verklaring in het register.
- Valt de website niet onder RVO? Pas dan het e-mailadres toegankelijkheid@rvo.nl aan naar het emailadres van de verantwoordelijke.

## Cookies

### Instructies

- Kopieer en plak de onderstaande tekst op de servicepagina. De URL is domein.nl/cookies.

      Een cookie is een bestand dat uw browser op uw computer opslaat. Bij een vervolgbezoek op onze website wordt u

## Archief

### Instructies

- Kopieer en plak onderstaande tekst op de servicepagina. Voor Nederlandse websites is de URL domein.nl/archief. Voor Engelse websites geldt domein.nl/archive.
- Plaats in de footer een link naar deze pagina met als linktekst 'Archief' of 'Archive'.
- In de Nederlandse tekst staat 'webarchief SITENAAM' en in de Engelse tekst 'SITE NAME web archive'. Link deze tekst naar het archief van de betreffende website. Pas 'SITENAAM/SITE NAME' aan naar de naam van de website. Gebruik als linktekst niet de url. Dat is niet goed leesbaar. De linktekst is dus bijvoorbeeld 'webarchief rvo.nl' of 'web archive rvo.nl'.
- De zin 'De zoekfunctie zoals u die op de website gewend bent, werkt niet in het webarchief.'is alleen van toepassing als de website een zoekbalk heeft. Laat de zin anders weg. Op de Engelse servicepagina geldt dat voor de zin 'The search function you're familiar with on the website does not work in the web archive.'
