# Paragraph

## Wanneer gebruik je een paragraph?

Je gebruikt de paragraph om een stuk tekst te tonen in een bepaalde stijl.

## Richtlijnen

- Houd de regellengte tussen de 55 en 75 tekens per regel. Dit is zorgt optimale leesbaarheid en begrip bij lopende tekst op een scherm.
