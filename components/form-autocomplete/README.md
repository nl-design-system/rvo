# Autocomplete

In de autocomplete wordt een lijst met suggesties weergegeven tijdens het typen. Ook worder er links weergegeven die naar andere onderdelen van een website of applicatie verwijzen.

## Wanneer gebruik je een Autocomplete

De autocomplete wordt voornamelijk in een zoekveld gebruikt.

## Anatomie van een Autocomplete

Een Autocomplete bestaat uit:

- Een textfield
- Een textfield die een lijst toont tijdens het typen
- Lijst met suggesties
- Lijst met links
